# Job Application Tracker 📊

A simple and user-friendly web application to track job applications and manage your job search efficiently.

## 🚀 Features
- Add job applications with company, role, and status
- Track progress (Applied, Interview, Rejected, Offer)
- Filter and search jobs
- Dashboard with application statistics
- Visual charts for better insights

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas

## ▶️ Run Locally
1. Clone the repository:
git clone https://github.com/harshinirhema/job-tracker-app

2. Install dependencies:
pip install -r requirements.txt

3. Run the app:
streamlit run app.py

## 📌 Use Case
Helps job seekers organize and track their applications instead of using spreadsheets.
